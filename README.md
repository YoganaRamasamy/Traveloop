# Traveloop – Smart Travel Planner

## Project Overview

Traveloop is a modern AI-powered travel planning web application designed to help users organize trips, manage budgets, explore destinations, and maintain travel journals in one platform.

This project was developed as a responsive front-end travel management system using HTML, Tailwind CSS, and JavaScript.

---

## Features

### Authentication System

* Login screen
* Email & password validation
* Password visibility toggle
* Remember me option
* Forgot password link
* Social login buttons (UI)

### Dashboard

* Attractive hero section
* Responsive navigation bar
* Travel statistics cards
* Smooth UI animations

### Trip Management

* Create new trips
